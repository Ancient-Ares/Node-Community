## node+express+ejs 制作简单社区后端

## 构建应用 
    express --view=ejs one_punch/     (one_punch 为文件名)
    npm install
    npm install express --save
    
## 启动
    node ./bin/www
    (NODE_ENV=production node ./bin/www 这是生产环境启动,没有错误提示...)
    (NODE_ENV=production PORT=8083 node ./bin/www 这是强制8083端口启动...)
